# Цифровой конструктор ПИН

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Flask](https://img.shields.io/badge/Flask-2.3+-green.svg)](https://flask.palletsprojects.com/)

Веб-приложение для автоматического формирования проектов установки информационных надписей и обозначений на объектах культурного наследия (памятниках истории и культуры) народов Российской Федерации.

##  Описание

**Цифровой конструктор ПИН** — специализированное программное обеспечение для собственников и правообладателей объектов культурного наследия Костромской области. Система позволяет в автоматическом режиме подготовить проект информационной таблички в соответствии с:

- Постановлением Правительства РФ от 10.09.2019 № 1178
- Требованиями к составу проектов установки и содержания информационных надписей
- ГОСТ Р 21.101-2020
- ГОСТ 2.304 (шрифт типа Б)

##  Возможности

### Для пользователей:
- ✅ **Автоматическое заполнение** — выбор объекта из реестра с автозаполнением полей
- ✅ **Интерактивный предпросмотр** — мгновенное отображение эскиза таблички
- ✅ **Динамический чертеж** — генерация чертежа пластины в масштабе 1:5
- ✅ **Автоматические расчеты** — расчет массы, ветровой нагрузки и несущей способности крепления
- ✅ **Редактирование изображений** — нанесение красных стрелок и рамок на схемы и фото
- ✅ **QR-коды** — автоматическая генерация QR-кода со ссылкой на объект в ЕГРОКН
- ✅ **Экспорт проекта** — скачивание готового проекта в форматах PDF и DOCX

### Для администраторов:
-  **Управление списком объектов** — загрузка и обновление базы объектов из Excel
-  **Редактирование справочников** — изменение вариантов заполнения полей
-  **Защищенная авторизация** — вход по логину и паролю

##  Установка

### Требования:
- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Git

### Шаги установки:

1. **Клонируйте репозиторий:**
```bash

```

2. **Создайте виртуальное окружение (рекомендуется):**
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

4. **Добавьте шрифт GOST_A.ttf:**
   
   Скачайте шрифт GOST Type B (GOST_A.ttf) и поместите его в корневую папку проекта рядом с `app.py`.
   
   > **Примечание:** Без этого шрифта PDF будет генерироваться со стандартным шрифтом.

5. **Запустите приложение:**
```bash
python app.py
```

6. **Откройте браузер:**
   
   Перейдите по адресу: http://127.0.0.1:5000

##  Использование

### Создание проекта:

1. **Выберите объект культурного наследия** из выпадающего списка
2. **Заполните обязательные поля** формы (помечены красной звездочкой *)
3. **Выберите размер таблички** (горизонтальная/вертикальная, малая/средняя/большая)
4. **Загрузите схему установки** (скриншот карты) и **фотофиксацию** объекта
5. **Нанесите красные маркеры** (стрелки или рамки) для указания места установки
6. **Нажмите "Сформировать проект"** — скачается ZIP-архив с PDF и DOCX файлами

### Административная панель:

1. Перейдите по ссылке **"Вход для администратора"** в футере сайта
2. Введите логин и пароль (по умолчанию: `admin` / `admin123`)
3. Загрузите Excel-файл со списком объектов (колонки: egrkn, name, address, history, fullName)
4. Нажмите **"Загрузить и обновить базу"**

##  Формат Excel-файла для загрузки объектов

Файл должен содержать следующие колонки:

| Колонка | Описание | Пример |
|---------|----------|--------|
| **egrkn** | Регистрационный номер в ЕГРОКН | 441410069670005 |
| **name** | Краткое наименование | Церковь Воскресения Словущего |
| **address** | Полный адрес | Костромская область, г. Кострома, ул. Симановского, д. 5 |
| **history** | Дата создания | 1685 г. |
| **fullName** | Полное название для таблички | Церковь Воскресения Словущего<br>1685 г. |
